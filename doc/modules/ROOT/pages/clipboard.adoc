These member functions are available for interacting
with the scheme component of URL containers:

[cols="a,a"]
|===
// Headers
|Name|Description

// Row 1, Column 1
|`has_scheme`
// Row 1, Column 2
|
// Row 1, Column 3
|Return `true` if a scheme is present.
// Row 1, Column 4
|

// Row 2, Column 1
|`scheme`
// Row 2, Column 2
|
// Row 2, Column 3
|Return the scheme as a string.
// Row 2, Column 4
|

// Row 3, Column 1
|`scheme_id`
// Row 3, Column 2
|
// Row 3, Column 3
|Return the scheme as a
    known scheme
    constant,
    `scheme::unknown`
    if the scheme is not well-known, or
    `scheme::none`
    if no scheme is present.
// Row 3, Column 4
|

// Row 4, Column 1
|`remove_scheme`
// Row 4, Column 2
|
// Row 4, Column 3
|Remove the scheme if present.
// Row 4, Column 4
|

// Row 5, Column 1
|`set_scheme`
// Row 5, Column 2
|
// Row 5, Column 3
|Set the scheme to a given string or 
    known scheme
    constant.
// Row 5, Column 4
|

|===
