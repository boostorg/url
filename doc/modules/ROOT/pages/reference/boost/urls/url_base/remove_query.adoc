== remove_query

pass:v,q[Remove the query]


=== Synopsis

=== Description

pass:v,q[If a query is present, it is removed.] pass:v,q[An empty query is distinct from having]
pass:v,q[no query.]

=== Example
[,cpp]
----
assert( url( "http://www.example.com?id=42" ).remove_query().buffer() == "http://www.example.com" );
----
