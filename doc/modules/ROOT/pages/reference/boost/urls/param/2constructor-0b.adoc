=== Description

pass:v,q[This constructs a parameter with a key] pass:v,q[and value.]
pass:v,q[No validation is performed on the strings.]
pass:v,q[Ownership of the key and value is acquired]
pass:v,q[by making copies.]

=== Example

----

----
[,cpp]
----

----
[,cpp]
----

----
[,cpp]
----

----
[,cpp]
----

----
